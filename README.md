# TCRT5000_experiment
Experiment using the TCRT5000 sensor using Arduino Uno.

This program reads the voltages of the LED and the Phototransistor using the Arduino's ananlog input pins.

Requirements:

  Hardware:
 
    Arduino Uno
    TCRT5000
    15k ohm resistor
    170 ohm resistor

  Software:

    Arduino IDE
